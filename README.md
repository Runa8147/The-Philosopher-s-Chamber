### Project Description
This project creates an engaging single-page web application simulating a philosophical debate between Aristotle and Socrates, powered by AI (Gemini & Gemma). Users enter a virtual museum environment, select a debate topic, and witness an AI-generated conversation, enhanced with animations and text-to-speech.
